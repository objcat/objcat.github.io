# 🍎 概念

## 🌲 什么是虚拟dom

想了解虚拟dom必须先知道真实dom, 在html中我们可以通过js去操作html元素, 比如修改值, 设置样式, 是因为html中有一颗真实的dom, 我们通过操作dom树就可以做到修改页面中静态的元素的目的

所谓`虚拟dom`就是把`html`中的元素使用`js对象`的形式表达出来, 当页面第一次渲染时会生成一颗虚拟dom树, 当页面再次进行改动时会生成一颗新树和老树通过diff算法进行比对, 然后把差异在真实dom中进行修改

它在简单的结构上可能没有直接使用dom操作效率高, 但是在复杂的dom操作上是有优势的, 可以防止页面频繁的做dom操作, 因为里面做了一些优化, 比如集中更新, 同时有很多处需要更新的地方, 会把这些修改在下一个事件循环中更新出来

## 🌲 diff算法

目的是做精细化比较从而可以让dom进行最小量的更新
