# 🍎 快速开始

## 🌲 起因

我打开了别人的项目后, 想运行项目出现了一个问题

```shell
System.IO.FileNotFoundException: Could not find file "D:\project\unity\zzjb2d\Unity\Assets\Bundles\Code\Unity.Model.dll.bytes"
```

经过多方询问, 这个东西就是`HybirdCLR`生成的, 所以我只能先学习一下这个
