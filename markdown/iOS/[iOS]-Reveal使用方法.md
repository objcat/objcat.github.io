####1.在官方网站下载软件 
http://revealapp.com
#####2.安装软件运行 找到以下栏目 如图1所示.
![](./images/3351b5a809bf5ec171441b26de38bb54/1.png)
####3.将下面的框架拖动到你的项目中 如图2所示.
![](./images/3351b5a809bf5ec171441b26de38bb54/2.png)

####5.在 build Phases -> link Binary With Libraries 添加 libz.tbd
